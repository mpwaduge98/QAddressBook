QAddressBook is a C++ applicaton made with QT Creator that allows the user to, 
     1. Create new contact 
     2. Edit contact
     3. Delete contact
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
* How to Create new Contact? 
-Type in User ID, FirstName, LastName, Email and PhoneNumber in the relevant fields and press "Save Contact". 
A message box will inform that the New Contact had been successfully saved. 

//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*Load Table Button
-The "Load Table" button will load a table onto the white space above it (TableView) and display the 
users table. 
-User can view all contacts in the table by loading the tabel onto the table view. 

//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*Counter Button
-By pressing the counter button user can view the total number of records in the 'users' table. 
 
//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*How to find a contact? 
-The search bar at the top of the table view allows the user to type in a ID number, FirstName, or LastName 
and returns the row that contains the contact information of the relevant ID, FirstName or LastName. 

//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*How to edit contact? 
-By double clicking on any of the rows in the table, you can retrieve the data into the relevant fields 
in User Details. 
-Once all data had been retireved to the relevant fields you can make the changes you wnat and 
click 'Save Contact"
//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*How to Delete contact?
-Double click on the row you want and click "Delete Contact". A message box will pop up indicating 
that the contact was deleted successfully. 
-If you want to add the contact back click on "Save Contact" and the entry will be successfully added 
back to the database. 
-Click the "Load Table" button again and the contact will be shown to be added back.
//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
*Export Button
-If you want to display the 'users' table in a text file click on the "Export" button and the table will be 
saved into a text file into a folder named "Files".
